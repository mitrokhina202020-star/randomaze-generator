# Randomaze Generator

PHP-инструмент для генерации рандомайзера учебных проектов, встраиваемого в GetCourse (или любую LMS с поддержкой HTML-блоков).

## Что это

Веб-приложение для технических специалистов онлайн-курсов. Принимает список проектов от методистов и превращает их в готовый HTML+JS-код рандомайзера для вставки в урок.

Студент видит кнопку «Получить проект» — рандомайзер выдаёт ему одно задание из списка. Количество попыток ограничено, каждый выбранный проект остаётся в истории, а финальный выбор сохраняется через webhook.

## Возможности

- **Редактор проектов** — добавление карточек с названием и rich-text описанием прямо в браузере
- **Импорт из Notion** — загрузка HTML-экспорта страницы с тоглами; каждый тогл становится отдельным проектом
- **Настройка параметров** — LESSON_ID, лимит попыток (MAX_ATTEMPTS), URL webhook для сохранения (SAVE_ENDPOINT)
- **Два варианта кода на выходе:**
  - Полный рандомайзер с описаниями, историей попыток и кнопкой «Сохранить»
  - Упрощённый список (только названия) с возможностью реролла
- **Самодостаточный файл** — один `.php` без внешних зависимостей и без базы данных

## Как работает сгенерированный виджет

```
Студент открывает урок
    ↓
Нажимает «Получить проект» → рандомайзер выбирает проект из списка
    ↓
Может перероллить (до MAX_ATTEMPTS раз), каждый выбор попадает в историю
    ↓
Нажимает «Сохранить» → POST-запрос на SAVE_ENDPOINT:
    { lesson_id, user_id, project_title, project_desc, attempts_left, timestamp }
```

Состояние хранится в `localStorage` под уникальным ключом на основе `LESSON_ID`.

## Требования

- PHP 7.4+ (для отдачи файла сервером)
- Либо просто открыть напрямую в браузере — файл работает без сервера

## Использование

1. Разместить `2026-06-15_randomaze-gen_script.php` на сервере (или открыть локально)
2. Заполнить **LESSON_ID**, **MAX_ATTEMPTS**, **SAVE_ENDPOINT**
3. Добавить проекты вручную кнопкой «+ Добавить проект» или загрузить Notion HTML-экспорт
4. Нажать **«Сгенерировать скрипт»**
5. Скопировать готовый код и вставить в HTML-блок урока в GetCourse

### Импорт из Notion

Экспортировать страницу из Notion: File → Export → HTML. Страница должна содержать **тоглы** — каждый тогл распознаётся как отдельный проект (заголовок тогла → название, содержимое → описание).

## Структура файла

Всё приложение — один файл:

| Слой | Описание |
|------|----------|
| HTML/CSS | Тёмный glassmorphism-интерфейс, адаптивный |
| JS — редактор | Карточки проектов с rich-text (`contenteditable`), toolbar форматирования |
| JS — импорт | Парсинг Notion HTML через `DOMParser`, маппинг тоглов в карточки |
| JS — генератор | `buildOutput()` и `buildSimpleOutput()` собирают готовый код построчно |
